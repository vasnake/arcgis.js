Справка к кнопке «Добавить» → «Добавить слой из библиотеки слоев».

В тулбаре карты вы можете видеть кнопку «Добавить», по нажатию на которую открывается меню. Эта кнопка и выпадающее меню дают возможность добавлять к карте разнообразные слои данных.

Меню «Добавить слой из библиотеки слоев».

При выборе этого пункта меню, в боковой панели карты открывается так называемая библиотека слоев. Сверху находится список выбора разделов (также известных как топики/topic). С помощью этого списка вы можете фильтровать набор показываемых в библиотеке слоев.

Все остальное место боковой панели занимает список слоев, зарегистрированных в библиотеке. Вы можете добавить любой из этих слоев к карте, щелкнув на изображении (тайл) выбранного слоя.
